# Customer Segmentation using K-Means Clustering

## Project Overview

This project uses Machine Learning to segment customers based on purchasing behavior.

The model was trained using K-Means Clustering and deployed through a Streamlit web application.

## Features

- Customer data analysis
- Data preprocessing and scaling
- K-Means clustering
- Customer segment prediction
- Interactive Streamlit web application

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Streamlit

## Files

- Analysis_Model.ipynb -> Data analysis and model training
- segmentation.py -> Streamlit application
- kmeans_model.pkl -> Trained clustering model
- scaler.pkl -> Data scaler
- marketing_campaign.csv -> Dataset
